# Symmetry Pattern Tool

Generates organic, symmetric ornamental patterns and frames from a name
(each name seeds its own pattern). Features quad (4-fold) mirror symmetry,
sharp flame/leaf stroke tips, and built-in illustration FX (bubble/blur,
wax, edge light) so patterns can be previewed, adjusted, and exported with
an image directly — no round-trip to Illustration software.

## Run
Open `index.html` in a browser, or serve the folder statically.

Type a name, pick a mirror mode (Quad = 4-fold symmetry), and tune the
stroke / flourish / FX controls.
